ngBolt is a component-based framework for rapid development of web applications
against a multi-platform architecture utilizing a single source code baseline

Rapid Development

ngBolt provides a CLI for creating, building, running, packaging, and deploying applications to multiple environments.

Lightweight Components

With a focus on extensibility and configuration, ngBolt provides common UI components and packages them on demand.

Application Logic

Since UI components and data access are included in ngBolt, the developer can focus on the application logic.

Consistent UI

ngBolt is based on Google's Material Design, which provides a consistent interface across many developers and languages.

Intuitive Design

By utilizing common components, animations, and layouts from ngBolt, developers can easily create intuitive application designs.

Platform Agnostic

Using the ngBolt Data API for data access abstraction with responsive UI layouts, a single baseline can deploy to many environments.

"Our goal is to provide an easy way for developers to
create, maintain, deploy, and build applications
for many different architectures"
