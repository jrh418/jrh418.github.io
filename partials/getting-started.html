Quick Start

The ngBolt CLI is a command line interface tool that can create a new project, add modules to existing projects, and aid development tasks such as testing, bundling, and deployment.

The goal of this quick start guide is to use the ngBolt CLI to build and run a simple application while adhering to the Style Guide.

By the end of the chapter, you should have a basic understanding of the CLI and be ready to move on to the tutorial to create a more complex application.

Before you begin: It is expected that ngBolt developers know and understand Angular JS (1.x). If you don't, please take the time to learn because it will not be explained in the ngBolt documentation.

Step 1. Set up your development environment

Before any development can be started, you need to set up your environment.

Install Node.js, npm, and git if they are not already on your machine.

If you are developing using Microsoft Windows, you will also need to install VC++ Build Tools and python 2.7.

Verify that you are running at least node 6.9.x, npm 3.x.x, and python 2.7.x by running node -v, npm -v, and python --version in a terminal/console window.
Windows users will also need to configure npm to use python and VC++ by running the following commands:
npm config set python python 2.7 npm config set msvs_version 2015 --global

Once these dependencies are configured, install the ngBolt CLI globally.

npm install -g ng-bolt-cli

Step 2. Create a new project

Open a terminal window and generate a new project and skeleton application by running the following command:

bolt new hello-world-app --template https://github.com/ngbolt/ng-bolt-example
Normally, you won't need to add the --template flag, but this will add a few extra examples to the application for you to review.

After executing this command, the following questions will appear for you to answer (defaults will be in parentheses):

Using template: https://github.com/ngbolt/ng-bolt-example What is the title of the project? This will be the title displayed in your application. (optional) Hello World What is the project version? (semver) (0.1.0) What is the project's description? (optional) Name of the company. (optional) List project contributors. (separate with commas. optional)
This process can take some time. The npm dependencies need to install and the template project will be cloned from the ngBolt Example Template.

Step 3. Run the application

Go to the project directory and launch the server.

cd hello-world-app bolt run

The bolt run command launches the server, watches your files, and rebuilds the app as you make changes to those files using gulp.

Open your web browser to http://localhost:9000/.

Your app should greet you with the login page:

You can log in to your application with the default credentials of admin / password. After you log in, you will see the landing page of the application:

Go ahead and click the "GET STARTED" button to load the home page.

Step 4. Make some changes

Now that the application is cloned and ready, we can make some common changes.

Let's start by changing the title on the app bar. Open the following file and replace the first section with the following:

ng-bolt-app/main/main-appbar.template.html 
<div class="appbar-section appbar-shrink">
    <img class="appbar-logo" src="images/logo.png">
    <h1 class="appbar-title">
        <!-- Update The Title Here -->
        <span>@@title</span>
    </h1>
</div>

As soon as you save the file, gulp will rebuild your app and you should now see the title of your application on the app bar.

The next step will be to use the configuration files to update the colors.

config/styles.json 
"primary-color": "$black",

Once saved, you should see the app bar has changed colors. Play around with the different colors and notice the color of the app bar text automatically updates based on the lighter / darker backgrounds.

Finally, we can update a view by modifying the following file:

ng-bolt-app/view1/view1.template.html 
<div class="grid-content grid-wrapper">
    <!-- Update The Page Title & Body Text -->
    <h1 class="font-title">Hello World!</h1>
    <p class="font-body">I've just updated my first ngBolt Application!</p>

    <!-- Add A User Form -->
    <form>
        <!-- The Form Has A Row -->
        <div class="form-row">
            <!-- Create A Text Field & Mark It Required -->
            <blt-textfield data-type="text" 
                        data-name="name"
                        data-model="Main.name" 
                        data-required="true"
                        data-label="Enter Your Name">
            </blt-textfield>

            <!-- Add A Submit Button With An Angular Function Callback -->
            <button class="btn-solid-submit" ng-click="Main.sayHello()">Go</button>
        </div>
    </form>

    <!-- Add A Divider -->
    <hr>

    <div class="btn-row">
        <button class="btn-solid" ng-click="Main.openView('/view2')">View 2</button>
        <button class="btn-solid" ng-click="Main.openView('/view3')">View 3</button>
    </div>
</div>

Add the appropriate function to the controller:

ng-bolt-app/main/main.controller.js 
// Within The `function MainController`
ctrl.getStarted = getStarted;
ctrl.openView = openView;
ctrl.logout = logout;

// Add This Extra Function
ctrl.sayHello = function() {
    if(ctrl.name) {
        console.alert("Hello " + ctrl.name + "!");
    }
}
