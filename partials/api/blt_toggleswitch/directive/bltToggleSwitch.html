blt_toggleswitch / bltToggleSwitch

The bltToggleSwitch directive can be used much like a standard HTML checkbox with the presentation of a switch with an 'on' and 'off' state. The toggle switch must be bound to a boolean attribute in your controller model. Selection/deselection of the toggle switch will toggle this value between true and false.

NOTE In order for the Toggle Switch to take input focus and support sequential keyboard navigation, you must add the tabindex="0" attribute on the directive. Documentation on using tabindex can be found here.
Best Practice The data- prefix is not required for ngBoltJS attributes, but is highly recommended to prevent conflicts with standard HTML attributes.

Attributes

Param Type Details
data-model expression

This attribute is used to bind the value of this directive to a property in the containing scope. Must be a boolean value.
data-disabled boolean

Disables the switch. A property in the containing scope that will disable the control if truthy. The Toggle Switch can be disabled in the off or on state.
data-label string

An optional value to display a form control label above the Toggle Switch.
data-justify string

An optional value to justify the Toggle Switch and label (if applicable) to the 'left' (default), 'right', or 'center'.

Examples

To use the Toggle Switch directive in your ngBoltJS application, include the blt-toggle-switch element in your HTML template. You can optionally include it inside a form. 

angular.module("bltDocs").controller("ToggleExCtrl1", function(){});
<div ng-controller="ToggleExCtrl1 as ctrl">
  <blt-toggle-switch data-model="ctrl.someBooleanValue"></blt-toggle-switch>
</div>

Usage inside form with option to disable. 

angular.module("bltDocs").controller("ToggleExCtrl2", function(){});
<div ng-controller="ToggleExCtrl2 as ctrl">
  <form class="form" novalidate name="toggleTest">
    <blt-toggle-switch data-model="ctrl.disableToggle"
                       data-label="Disable">
    </blt-toggle-switch>
    <blt-toggle-switch data-model="ctrl.someBooleanValue"
                       data-disabled="ctrl.disableToggle">
    </blt-toggle-switch>
  </form>
</div>

With a label.

Note You can bind Angular expressions to your label. 
angular.module("bltDocs").controller("ToggleExCtrl3", function(){});
<div ng-controller="ToggleExCtrl3 as ctrl">
  <blt-toggle-switch data-model="ctrl.someBooleanValue"
                     tabindex="0"
                     data-label="Toggle Switch: {{ctrl.someBooleanValue}}">
  </blt-toggle-switch>
</div>

Justify right. 

angular.module("bltDocs").controller("ToggleExCtrl4", function(){});
<div ng-controller="ToggleExCtrl4 as ctrl">
  <blt-toggle-switch data-model="ctrl.someBooleanValue"
    data-justify="right"
    tabindex="0"
    data-label="Right Justified">
  </blt-toggle-switch>
</div>
