Service components in blt_auth

Name Description
BltAuth

This service provides an API to facilitate user authentication in ngBoltJS applications.
BltAuthStorageService

Storage API for stashing and retrieving login tokens/cookies.
DevAuthService

This is a development version of the internal authentication service that can be used to provide user authentication through the BltAuth API during development and testing.
WampAuthService

This version of the internal authentication service is built to connect to a WAMP router that exposes login related RPC endpoints.
