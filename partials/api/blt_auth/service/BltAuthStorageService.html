blt_auth / BltAuthStorageService

Storage API for stashing and retrieving login tokens/cookies.

If cookie storage is available, key/value pairs will be stored as cookies. Otherwise window.localStorage will be used.

Public Methods

  • retrieve trending_flat String
  • store
  • remove

Dependencies

  • $window

Methods

retrieve(key);

Retrieves the value assigned to the given key.

Parameters

Param Type Details
key String

The key to retrieve the value for.

Returns

String

The value assigned to the given key, if found, undefined otherwise.

store(key, value, duration);

Stores the given value in the given key for the given duration.

Parameters

Param Type Details
key String

The key to assign the value to.
value String

The value to store.
duration Number

The duration in minutes to store the given value. -1 will cause value to never expire.

default: 60

remove(key);

Clears the value from the given key.

Parameters

Param Type Details
key String

The key to clear.
