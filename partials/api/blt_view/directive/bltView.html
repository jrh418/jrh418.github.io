blt_view / bltView

since 1.11.0

This directive is a wrapper for Angular's ngView directive, which contains the 'pages' of your application and applies CSS classnames for the animation that was set in your app's views.json file.

The current view follows the path of the URL which can be changed by using the href attribute of an a tag or the ng-href directive. You can change views programmatically using the Angular $location service or by passing the routes and params to BltApi.

See blt_appViews for documentation on how to define views.

Basic Usage

NOTERestrict directive to an HTML Element tag.

<blt-view></blt-view>
