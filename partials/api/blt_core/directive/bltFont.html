blt_core / bltFont

since 1.0.0

Defines basic styles for fonts in ngBoltJS applications. ngBoltJS uses Roboto as its default font-family but falls back to Helvetica, Arial, and sans-serif.

To use ngBoltJS Font styles in your ngBoltJS application, include the following class elements and modifiers in your HTML templates.

CSS Classes

Param Type Details
font-title element

Default title font. Can be used on any HTML heading tag <h1> ... <h6>, but would normally be found on an <h1> or <h2> tag. Do not use on component titles as they have their own specific styles.
font-sub-title element

The default sub-title font. Should be used after a .font-title but can be used on any text element for stylistic purposes. Is normally found on <h2> and <h3> tags but can be used on <span> tags as part of a paragraph of text.
font-body element

The default text for ngBoltJS applications. Should be used on <p> and <span> tags.
font-caption element

Use for smaller caption sized fonts. Should be used on <p> and <span> tags.
font-icon element

Use on icons throughout your application. These are icons that are for display purposes only. If to be used on a button, use .btn-icon instead.
font-xlarge modifier

Use on any font element child HTML tag to increase the font size by 2%. Must be used on a nested HTML tag of the containing font element.
font-large modifier

Use on any font element child HTML tag to increase the font size by 1.5%. Must be used on a nested HTML tag of the containing font element.
font-small modifier

Use on any font element child HTML tag to decrease the font size by .875%. Must be used on a nested HTML tag of the containing font element.
font-tiny modifier

Use on any font element child HTML tag to decrease the font size by .75%. Must be used on a nested HTML tag of the containing font element.
font-bolder modifier

Use on any font element child HTML tag to increase the font weight. Must be used on a nested HTML tag of the containing font element.
font-bolder-'bp'-'dir' modifier

'font-bolder' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
font-lighter modifier

Use on any font element child HTML tag to decrease the font weight. Must be used on a nested HTML tag of the containing font element.
font-lighter-'bp'-'dir' modifier

'font-lighter' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
font-code modifier

Use on any font element to display as monospace code with a light background to offset it from it's surrounding content.
font-error modifier

Use on any font element to color it red.
font-warning modifier

Use on any font element to color it amber.
font-success modifier

Use on any font element to color it green.
