blt_core / bltCssUtilities

ngBoltJS comes bundled with a few utility classes that can be applied to several components throughout your application.

To use a utility class, simply add it to the desired HTML tag in your application. Several of these utility classes override default settings and should be placed as the last classname in the class attribute.

CSS Classes

Param Type Details
spacer element

Adds a horizontal space between elements. Must use on an HTML <hr> tag.
spacer-vertical element

Adds a vertical space between elements. Must use on an HTML <div> tag.
edge-top element

Adds a dark border to the top of the respective element.
edge-top-'bp'-'dir' element

'edge-top' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range of (up|down|only).
edge-right element

Adds a dark border to the right of the respective element.
edge-right-'bp'-'dir' element

'edge-right' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
edge-bottom element

Adds a dark border to the bottom of the respective element.
edge-bottom-'bp'-'dir' element

'edge-bottom' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
edge-left element

Adds a dark border to the left of the respective element.
edge-left-'bp'-'dir' element

'edge-left' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
edge-*-light element

Adds a light border to the specified edge where * is (top|right|bottom|left) of the respective element.
edge-*-light-'bp'-'dir' element

'edge-*-light' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
full-screen-bg element

An overlay that covers the entire window of the application and includes a message and optionally an action button. Should be used on an HTML <div> tag.
full-screen-msg element

The message in a .full-screen-bg. Should be used on an HTML <p> tag.
collapse modifier

Removes all padding from an element.
collapse-'bp'-'dir' modifier

'collapse' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
wrapper modifier

Adds 1rem (16px) of padding around an element.
wrapper-'bp'-'dir' modifier

'wrapper' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
shrink modifier

Shrinks the width of the element to the width of its content. Can only be used on elements with display property of 'flex'.
shrink-'bp'-'dir' modifier

'shrink' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
text-center modifier

Center align text. Does not work on elements with a display property of 'flex'.
text-center-'bp'-'dir' modifier

'text-center' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
text-left modifier

Left align text. Does not work on elements with a display property of 'flex'.
text-left-'bp'-'dir' modifier

'text-left' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
text-right modifier

Right aligned text. Does not work on elements with a display property of 'flex'.
text-right-'bp'-'dir' modifier

'text-right' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
justify-center modifier

Center justify children elements. Only works on elements with a display property of 'flex'.
justify-center-'bp'-'dir' modifier

'justify-center' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
justify-left modifier

Left justify children elements. Only works on elements with a display property of 'flex'.
justify-left-'bp'-'dir' modifier

'justify-left' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
justify-right modifier

Right justify children elements. Only works on elements with a display property of 'flex'.
justify-right-'bp'-'dir' modifier

'justify-right' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
overflow-visible modifier

Make overflowed content visible and prevent scrolling.

Examples

Common Use Cases 

<!-- Add a border to a panel footer -->
<footer class="panel-footer edge-top"> ... </footer>

<!-- Add a border between a menu and a view -->
<div class="grid-block">
    <div class="grid-block grid-shrink">
        <ul class="menu edge-right"> ... </ul>
    </div>
    <div class="grid-content view-container>
        <blt-view> ... <blt-view>
    </div>
</div>

<!-- Add a horizontal space between elements in a panel-body -->
<div class="panel-body">
    <p class="panel-text panel-wrapper">Some instructions</p>
    <hr class="spacer">
    <ul class="menu> ... </ul>
</div>

<!-- Add a loading message on app start up -->
<div class="full-screen-bg" ng-if="Main.loading">
    <p class="full-screen-msg">{{Main.configuringMessage || 'Configuring Application...'}}</p>
    <p class="full-screen-msg"><span class="font-tiny">Step {{Main.currentStep}} of {{Main.totalSteps}}</p>
    <button class="btn-solid-submit" ng-click="Main.open()" ng-show="Main.loadingComplete">Enter</button>
</div>

Using Media Queries

See the Media Queries Guide for more information on using media queries. 

<!-- Add a border to a panel footer for medium screens and larger -->
<footer class="panel-footer edge-top-md-up"> ... </footer>

<!-- Shrink the grid for small screens only -->
<div class="grid-block">
    <div class="grid-block grid-shrink-sm-only">
        ...
    </div>
</div>

<!-- Add a horizontal space between elements in a panel-body -->
<div class="panel-body">
    <!-- Wrapper for large screens and bigger -->
    <p class="panel-text panel-wrapper-lg-up">Some instructions</p>
    <hr class="spacer">
    <ul class="menu"> ... </ul>
</div>
