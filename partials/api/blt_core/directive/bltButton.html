blt_core / bltButton

since 1.0.0

The ngBoltJS Button component is used for triggering actions within an ngBoltJS application. The presentation of the button can be customized through the class elements and modifiers supported by this component.

To use the Button component in your ngBoltJS application, simply include the relevant classes on an HTML <button> tag in your application and use the ng-click to handle the button click event.

CSS Classes

Param Type Details
btn-text element

Default text button for secondary actions. Must be used on an HTML <button> tag.
btn-text-'bp'-'dir' element

'btn-text' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
btn-solid element

A text button with a color background used for secondary actions. Must be used on an HTML <button> tag.
btn-solid-'bp'-'dir' element

'btn-solid' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
btn-icon element

An icon button. Must be used on an HTML <button> tag.
btn-icon-'bp'-'dir' element

'btn-icon' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
btn-link element

A button that looks similar to hyperlinks without an underline. Use on an HTML <a> tag or <button> tag.
btn-link-'bp'-'dir' element

'btn-link' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
btn-row element

A container for displaying buttons horizontally and justifies the buttons to the right by default. Use on an HTML <div> tag.
btn-text-submit modifier

Colored, primary text button to use for primary actions. Must be used on an HTML <button> tag.
btn-solid-submit modifier

Colored, primary solid button to use for primary actions. Must be used on an HTML <button> tag.
btn-block modifier

Make a button the full width of its containers. Useful in modals, panels, or forms. Must be added to a .btn-** element.
btn-row-left modifier

Left justify the content of a .btn-row element.
btn-row-left-'bp'-'dir' modifier

'btn-row-left' with media query support for various screen sizes where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
btn-row-center modifier

Center justify the content of a .btn-row element.

Examples

Common Use Cases 

<!-- Text Buttons -->
<button class="btn-text" blt-close="">Cancel</button>
<button class="btn-text-submit" ng-click="Main.submit()">Submit</button>

<!-- Solid Buttons -->
<button class="btn-solid" blt-close="">Canel</button>
<button class="btn-solid-submit" ng-click="Main.submit()">Submit</button>

<!-- Icon Button -->
<button class="btn-icon" ng-click="Main.approve()"><span class='fa fa-check-circle'></span></button>

<!-- Link Buttons -->
<button class="btn-link" ng-click="Main.edit()">Edit</button>
<a class="btn-link" href="http://some/url">Open</a>

<!-- Row of Buttons -->
<div class="btn-row">
  <button class="btn-solid" blt-close="">Canel</button>
  <button class="btn-solid-submit" ng-click="Main.submit()">Submit</button>
</div>

Disabled Buttons

To disable a button, use an expression with ng-disabled to put the button in a disabled state. Disabled buttons will gray out and the cursor will be set to default.

<button class="btn-text-submit" ng-click="Main.submit()" ng-disabled="Main.formError == true">Submit</button>

Using Media Queries

See the Media Queries Guide for more information on using media queries.

<!-- Text Button with Media Query -->
<!-- For screens size medium and smaller-->
<button class="btn-text-md-down" blt-close="">Cancel</button>

<!-- Solid Button with Media Query -->
<!-- For screens size large and smaller-->
<button class="btn-solid-lg-down" blt-close="">Cancel</button>

<!-- Icon Button with Media Query -->
<!-- For small screens only -->
<button class="btn-icon-sm-only" ng-click="Main.approve()"><span class='fa fa-check-circle'></span></button>

<!-- Link Button with Media Query -->
<!-- For screens size medium and larger -->
<button class="btn-link-md-up" ng-click="Main.edit()">Edit</button>
