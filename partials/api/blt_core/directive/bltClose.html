blt_core / bltClose

Closes modals and panels. To close a modal or panel, pass the id of the modal or panel as the value of the directive. This directive should be used on an ngBoltJS button in place of an ng-click.

Note If other functions need to be triggered when closing a component, use ng-click. Call a function in your controller and use BltApi to close the component instead.
Tip If the close button is a child element of a modal or panel, you do not need to pass the id of the modal or panel.

Dependencies

  • BltApi

Examples

Close a panel with an id of 'myPanel' 

<button class="btn-text" blt-close="myPanel">Close Panel</button>

Close panel from within panel 

<blt-panel>
...
<button class="btn-text" blt-close="">Close</button>
...
</blt-panel>
