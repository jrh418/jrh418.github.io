blt_core / bltOpenView

Use the href attribute or ngHref directive on an HTML <a> tag instead.

Opens a new view and closes the current one. To open a view, pass the path of the view as the value of the directive. This directive should be used on an ngBoltJS button in place of an ng-click.

Note If other functions need to be triggered before opening views, or if you need to pass other paramaters to the route, use ng-click. Call a function in your controller and use BltApi.

Dependencies

  • BltApi

Attributes

Param Type Details
path string

The string of the path to pass to BltApi.

Example

Open a view with a path of '/my-view' 

<button class="btn-text" blt-open-view="/my-view">Open View</button>
