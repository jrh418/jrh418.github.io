blt_core / bltOpen

Use blt-open to open modals and directives. To open a modal or panel, pass the id of the modal or panel as the value of the directive. This directive should be used on an ngBoltJS button in place of an ng-click.

Note If other functions need to be triggered when closing a component, use ng-click. Call a function in your controller and use BltApi to open the component instead.

Dependencies

  • BltApi

Example

Open a modal with an id of 'myModal' 

<button class="btn-text" blt-open="myModal">Open Modal</button>
