blt_menu / bltMenu

since 1.0.0

The bltMenu component is primarily used for navigation within the application or for opening and closing different sections of the application. It is displayed as a vertical list of buttons and can contain sub-menus. The presentation of the menu options can be customized through the class elements and modifiers supported by this component.

To use the bltMenu component in your ngBoltJS application, include the relevant class elements inside a HTML <ul> element in your application and use the ng-click directive to complete some action.

Basic Usage

NOTERestrict directive to an HTML Element tag Element tag or Attribute Attribute.

Minimal Usage

<ul class="menu">
 <li class="menu-item">
     <button class="menu-link" ng-click="Main.openItem('item1')">Item 1</button>
 </li>
 <li class="menu-item">
     <button class="menu-link" ng-click="Main.openItem('item2')">Item 2</button>
 </li>
 <li class="menu-item">
     <button class="menu-link" ng-click="Main.openItem('item3')">Item 3</button>
 </li>
</ul>

CSS Classes

Param Type Details
menu element

The main containing element. Should be placed on the HTML <ul> tag. Can be modified with: .menu–thin, .menu–wide, .menu–condensed, .menu–borders, .menu–icons, .menu–accordion, .menu–floating, .menu-hide-active.
menu-item element

The menu list item. Should be used on the HTML <li> tag. Contains the .menu-link element and, optionally, another menu in an accordion menu. Can be modified with .menu-active.
menu-link element

The actionable content of a .menu-item content. Should be used on the HTML <button> tag and must be the child of a .menu-item element. Use an Angular expression on the ng-click directive to trigger the action.
menu-icon element

An optional icon to add to the text of a .menu-link element. Must be the child of a .menu-link and should be used on the HTML <span> tag.
menu-text element

The text of the .menu-link element. Is only required when a .menu-icon is also present. Must be a child of a .menu-link element and should be used on a HTML <span> tag.
menu-divider element

A horizontal border useful for dividing groups of .menu-items. Should be used on an HTML <hr> tag.
menu-container element

A required wrapper for floating menus so the menu is positioned correctly. Should contain a button element for toggling the menu and a floating menu.
menu-thin modifier

Narrows the default width of a menu component to 8rem (128px). Add to the .menu element.
menu-wide modifier

Widens the default width of a menu component to 15rem (240px). Add to the .menu element.
menu-condensed modifier

Lessens the padding of .menu-item elements to condense the height of the menu. Add to the .menu element.
menu-condensed-'bp'-'dir' modifier

menu-condensed with media query support for various screen sizes (see example) where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
menu-borders modifier

Adds a border to each .menu-item element and thickens the width of a .menu-divider border. Add to the .menu element.
menu-borders-'bp'-'dir' modifier

menu-borders with media query support for various screen sizes (see example) where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
menu-icons modifier

Narrows the width of a menu to 3rem (48px). Requires that all .menu-link elements use icons only and no text. Add to the .menu element.
menu-icons-'bp'-'dir' modifier

menu-icons with media query support for various screen sizes (see example) where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
menu-accordion modifier

Creates a collapsing set of menus. Sub menus have a darker background to set them apart from their parent .menu-items. Each .menu-item should contain another menu. Use the ng-show and ng-hide directives to show/hide the sub menus. Add to the .menu element.
menu-accordion-'bp'-'dir' modifier

menu-accordion with media query support for various screen sizes (see example) where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
menu-floating modifier

Creates an absolutely positioned menu like a simple dropdown. Use the ng-show and ng-hide directives to show/hide the floating menu and toggle with a .btn-text or .btn-solid component. Add to the .menu element. Must be the child of a .menu-container element.
menu-floating-'bp'-'dir' modifier

menu-floating with media query support for various screen sizes (see example) where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).
menu-left modifier

Align a floating menu to the left. Add to a .menu .menu-floating element.
menu-right modifier

Align a floating menu to the right. Add to a .menu .menu-floating element.
menu-active modifier

Add this class to the selected .menu-item using ng-class. Highlights the selected .menu-item with the $btn-link-color (blue by default). Add to the .menu-item element.
menu-hide-active modifier

Use this modifier to hide the active .menu-item elements. Add to the .menu element.
menu-hide-active-'bp'-'dir' modifier

menu-hide-active with media query support for various screen sizes (see example) where bp is the breakpoint (sm|md|lg|xl|xxl) and dir is the range (up|down|only).

Examples

Triggering Actions

Use the ng-click directive to trigger actions when clicking a .menu-link. Additionally, use the ng-class directive to add the .menu-active class to the respective .menu-item when that item is selected. Lastly, you can disable a menu link by passing some expression to the ng-disabled directive.

angular.module("bltDocs")
  .controller("MenuExCtrl", function(){
    var ctrl = this;
    ctrl.disableItem4 = true;
  });
<!-- This link will be highlighted when clicked -->
<ul class="menu" ng-controller="MenuExCtrl as ctrl">
  <li class="menu-item" ng-class="{'menu-active': ctrl.currentItem == 'item2'}">
    <button class="menu-link" ng-click="ctrl.currentItem = 'item2';">Item 2</button>
  </li>
  <li class="menu-item" ng-class="{'menu-active': ctrl.currentItem == 'item3'}">
    <button class="menu-link" ng-click="ctrl.currentItem = 'item3';">Item 3</button>
  </li>
  <!-- This link will be disabled when the ng-disabled expression is true -->
  <li class="menu-item">
    <button class="menu-link" ng-click="ctrl.currentItem = 'item4';" ng-disabled="ctrl.disableItem4">Item 4</button>
  </li>
</ul>

Menu Sizes

Menus have three sizes: Default (200px), Thin (128px) and Wide (240px). Use the .menu-thin or .menu-wide to adjust the size of the menu.

<!-- Thin Menu (128px) -->
<ul class="menu menu-thin"> ... </ul>

<!-- Wide Menu (240px) -->
<ul class="menu menu-wide"> ... </ul>

Custom Width

To manually set the width of the menu, add an id to your .menu element and set the width in a .scss or a .css file.

#customMenu{
  // Set the menu width to 400px;
  width: 25rem; //400px
}
<ul id="customMenu" class="menu"> ... </ul>

Options

Menus have multiple display options including: .menu-borders, .menu-condensed, .menu-icons, .menu-hide-active.

<!-- Menu with borders between each menu item -->
 <ul class="menu menu-borders"> ... </ul>

 <!-- Menu with less padding between each item -->
 <ul class="menu menu-condensed"> ... </ul>

 <!-- Menu with only icons for .menu-links -->
 <ul class="menu menu-icons">
 <li class="menu-item">
 <button class="menu-link">
 <span class="menu-icon fa fa-home"></span>
 </button>
 </li>
 </ul>

 <!-- Hide active menu items -->
 <ul class="menu menu-hide-active"> ... </ul>

 <!-- Include Icons with a menu-link -->
 <button class="menu-link">
 <span class="menu-icon fa fa-home"></span>
 <span class="menu-text">Home</span>
 </button>

Grouping Menu Content

You can make groups of .menu-items by using a .menu-divider on an empty HTML <li> tag.

<ul class="menu">
 <li class="menu-item"> ... </li>
 <li class="menu-item"> ... </li>

 <!-- use a divider to group menu items -->
 <li class="menu-divider"></li>

 <li class="menu-item"> ... </li>
 <li class="menu-item"> ... </li>
 </ul>

Accordion Menu

Accordion menus contain at least two layers of navigation. Simply add another .menu element inside a .menu-item and use ng-click, ng-show and ng-hide to toggle the sub-menu.

<ul class="menu menu-accordion">
 <li class="menu-item">
 <button class="menu-link" ng-click="Main.currentMenu = 1;">Menu 1</button>
 <ul class="menu" ng-show="Main.currentMenu == 1"> ... </ul>
 </li>
 <li class="menu-item">
 <button class="menu-link" ng-click="Main.currentMenu = 2;">Menu 2</button>
 <ul class="menu" ng-show="Main.currentMenu == 2"> ... </ul>
 </li>
 </ul>

Floating Menu

Use floating menus combined with a toggle button to create a simple dropdowns anywhere in your application. (For a more robust dropdown use bltDropdown).

<div class="menu-container">
 <button class="btn-solid" ng-click="Main.showDropdown = !Main.showDropdown">Toggle Menu</button>

 <!-- Use .menu-left to align the floating menu to the left. -->
 <ul class="menu menu-floating menu-left" ng-show="Main.showDropdown == true"> ... </ul>

 <!-- Use .menu-right to align the floating menu to the right. -->
 <ul class="menu menu-floating menu-right" ng-show="Main.showDropdown == true"> ... </ul>
 </div>

Using Media Queries

See the Media Queries Guide for more information on using media queries. 

<!-- Condensed menu for screens medium size and smaller -->
<ul class="menu menu-condensed-md-down"> ... </ul>
<!-- Menu with borders between each item for small screens only -->
<ul class="menu menu-borders-sm-only"> ... </ul>
<!-- Menu with only icons for screens medium size and larger -->
<ul class="menu menu-icons-md-up"> ... </ul>
<!-- Accordion menu for xxl screens only -->
<ul class="menu menu-accordion-xxl-only"> ... </ul>
<!-- Floating menu for screens large and smaller -->
<ul class="menu menu-floating-lg-down"> ... </ul>
