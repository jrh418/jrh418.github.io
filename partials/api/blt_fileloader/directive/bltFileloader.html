blt_fileloader / bltFileloader

The bltFileloader provides simple file selection capability. Upon selection, a JavaScript File object representing the selected file will be set into the controller model specified in the data-model attribute.

BEST PRACTICE The data- prefix is not required for ngBoltJS attributes, but is highly recommended to prevent conflicts with standard HTML attributes.

Dependencies

  • BltApi

Attributes

Param Type Details
data-name string

This attribute indicates the name of this form element and will be used during form traversal by the ngBoltJS framework.
data-model expression

This attribute is used to bind the value of this component to a property in the containing scope. Functionality is based on the Angular ngModel directive.
data-disabled boolean

Disables the field. Any value set in this attribute will cause the field to be disabled.
data-autofocus boolean

Indicates whether or not this field should autofocus on page load.
data-required value

Indicates whether or not this field is required.
data-tabindex value

Specifies the tab order of an element

Example

Basic Usage 

angular.module('bltDocs')
  .controller("FileExCtrl", function(){
    var ctrl = this;
    ctrl.file = undefined;
  });
<div ng-controller="FileExCtrl as ctrl">
  <form name="ctrl.myForm" class="form" novalidate>
      <blt-fileloader data-name="fileloader"
                      data-model="ctrl.file">
      </blt-fileloader>
  </form>
</div>
