ngBoltJS / blt_view

ngBoltJS View component.

Module Components

Directive

Name Description
bltView

This directive is a wrapper for Angular's ngView directive, which contains the 'pages' of your application and applies CSS classnames for the animation that was set in your app's views.json file.

Service

Name Description
ViewFactory

This factory provides basic view property information such as the current view's index and state, as well as what animation classes are attached to it.
