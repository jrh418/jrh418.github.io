blt_dropdown / bltDropdown

The bltDropdown component is used to collect input from the user from a list of known values. This is an extension of the HTML <select> element. The presentation of the dropdown options can be customized through the attributes supported on this directive.

To use the bltDropdown component in your ngBoltJS application, you simply need to include a blt-dropdown element inside of a form in your application.

Best Practice The data- prefix is not required for ngBoltJS attributes, but is highly recommended to prevent conflicts with standard HTML attributes.

Dependencies

  • BltApi
  • $timeout
  • $rootScope.Scope

Attributes

Param Type Details
data-label string

This attribute specifies the label for this component.
data-model expression

This attribute is used to bind the value of this component to a property in the containing scope. Functionality is based on the Angular ngModel directive.
data-name string

This attribute indicates the name of this form element and will be used during form traversal by the ngBoltJS framework.
data-options expression

This attribute is used to bind to a list of options that will be presented to the user. This can either be an array of options or a model of value/label pairs to use as options.
data-autofocus boolean

Indicates whether or not this field should autofocus on page load.
data-change expression

This attribute is used to bind an expression in the containing scope that will be invoked any time the value of this component changes. Functionality is based on the Angular ngChange directive.
data-required boolean

Indicates whether or not this field is required.
data-type string

Specifies the dropdown type (dropdown, select, or searchable).
data-tabindex value

Specifies the tab order of an element.
data-disabled boolean

Disables the field. Any value set in this attribute will cause the field to be disabled.

Examples

Standard select dropdown. The user will be presented with the options contained in the arrayOfOptions. 

angular.module("bltDocs")
  .controller("DropdownExampleCtrl", function(){
    var ctrl = this;
    ctrl.arrayOfOptions = ["Value 1", "Value 2", "Value 3"];
  });
<div ng-controller="DropdownExampleCtrl as ctrl">
  <form name="ctrl.myForm" class="form" novalidate>
    <blt-dropdown data-type="select"
                  data-name="dropdown1"
                  data-label="Select a Value"
                  data-options="ctrl.arrayOfOptions"
                  data-model="ctrl.dropdown1">
    </blt-dropdown>
  </form>
</div>

Searchable dropdown. The user will be presented with the options contained in the arrayOfOptions. When the list of options is opened for selection, the user will be able to type into the input field to filter available options. 

angular.module("bltDocs")
  .controller("DropdownExampleCtrl2", function(){
    var ctrl = this;
    ctrl.arrayOfOptions = ["Value 1", "Value 2", "Value 3"];
  });
<div ng-controller="DropdownExampleCtrl2 as ctrl">
  <form name="ctrl.myForm" class="form" novalidate>
    <blt-dropdown data-type="searchable"
                  data-name="dropdown2"
                  data-label="Select a Value"
                  data-options="ctrl.arrayOfOptions"
                  data-model="ctrl.dropdown2">
    </blt-dropdown>
  </form>
</div>

Dropdown with "dropdown" style. This will present the options contained in the optionMap as a clickable dropdown link. The presentation is similar to that of a dropdown menu. 

angular.module("bltDocs")
  .controller("DropdownExampleCtrl3", function(){
    var ctrl = this;
    ctrl.optionMap = {
      value1: "Value One",
      value2: "Value Two",
      value3: "Value Three"
    }
  });
<div ng-controller="DropdownExampleCtrl3 as ctrl">
  <form name="ctrl.myForm" class="form" novalidate>
    <blt-dropdown data-type="dropdown"
            data-name="dropdown3"
            data-label="Select a Value"
            data-options="ctrl.optionMap"
            data-model="ctrl.select3">
    </blt-dropdown>
  </form>
</div>
